Firefox Extension
PROJECT: HERALD
Automated order entry for the Pet Valu supplier portal. Runs directly in your Firefox sidebar — open it, hit Start, walk away.
Install Extension
Requires Firefox Developer Edition. Don't have it yet? Follow Steps 1–3 on the right first.
Updating the extension? Skip all steps — just click Install Extension above and Firefox will handle the rest.
First-time setup — follow these steps in order
Already set up before?  Skip straight to Install Extension on the left.
1
Download Firefox Developer Edition
Project: Herald only works in Firefox Developer Edition — the regular version won't work.

→ firefox.com/en-CA/channel/desktop/developer/

Download and install it, then open Firefox Developer Edition before continuing.
2
Open the advanced settings page
In the Firefox Developer Edition address bar, type exactly this and press Enter:

about:config

A warning page will appear — click "Accept the Risk and Continue".
3
Allow the extension to be installed
In the search bar at the top of that page, search for:

xpinstall.signatures.required

One result will appear. Double-click it to flip its value from true to false.

Done — you can close this tab.
4
Install Project: Herald
Click Install Extension on the left. A Firefox popup will appear — click Add.
5
Open the sidebar
In Firefox, click:

View → Sidebar → Project: Herald

The Project: Herald panel will open on the right side of your browser. You're all set.
